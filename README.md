### 배운 점(WHAT I LEARN)

- eval 함수의 활용법을 알았다.
- 특정 문자를 클릭했을 시 문서에 보이게 하는 방법을 알았다.
- align-items: stretch를 활용해 보았다.
- 반복문 사용 시 힘든 부분이 있었는데, 반복적으로 자주 사용해 봐야겠다.
